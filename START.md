# 🚀 Cómo iniciar NeoMonitor Expert Suite

## Comando rápido

```bash
cd /home/diegosteinberg/generador && ./start.sh
```

O sin el script:

```bash
cd /home/diegosteinberg/generador && node server.js
```

## Luego

1. Abre en tu navegador: 
   ### **[http://localhost:8000](http://localhost:8000)**
2. Selecciona el módulo que quieras usar
3. Para generar casos con IA:
   - Ve a **Casos Expandidos**
   - Busca **"Generar caso con IA"**
   - Escribe la descripción del caso
   - Cliquea **"Generar Caso"**
   - Ingresa tu API key de Claude la primera vez

## Para detener

En la terminal: `Ctrl+C`

---

**Todo está en un único servidor.** No necesitas abrir múltiples terminales.
